<p>Today is one of those awesome days if you are building stuff on .NET platform. Microsoft announced bunch of stuff at <a href="http://channel9.msdn.com/Events/Build/2015">Build 2015</a> keynote a few hours ago and one of them is <a href="https://code.visualstudio.com">Visual Studio Code</a>, a free and stripped down version of Visual Studio which works on <a href="https://code.visualstudio.com/Docs/setup#_mac-os-x">Mac OS X</a>, <a href="https://code.visualstudio.com/Docs/setup#_linux">Linux</a> and <a href="https://code.visualstudio.com/Docs/setup#_windows">Windows</a>. It leverages bunch of existing open source software like <a href="http://www.omnisharp.net/">OmniSharp</a>, <a href="https://github.com/atom/electron">Electron</a>.
